# Hermes Agent

Ce repo contient ma documentation sur l'agent Hermes.

## Introduction

Hermes est un agent IA développé par Nous Research. Tout comme OpenClaw il est
capable d'agir, de coder, d'exécuter des scripts, de faire des recherches sur
le Web, de communiquer via de multiples canaux (Email, Telegram, Discord, etc.).
Il intègre une boucle d'auto apprentissage et d'amélioration. Il est également
capable de développer ses propres skills et de les affiner. Pour en savoir plus
je mets les liens officiels ci-dessous.
